# Euromoon Project

## Beschrijving
Deze applicatie is een command-line Java programma waarmee passagiers geregistreerd worden, reizen worden aangemaakt, treinen gekoppeld worden en tickets verkocht kunnen worden.
## Functionaliteiten
- Registreren van passagiers  
- Aanmaken van reizen  
- Koppelen van een trein aan een reis  
- Verkopen van tickets  
- Afdrukken van een boardinglijst naar een tekstbestand
- ## Structuur
De code is opgesplitst in duidelijke packages:
- entity.person → Person, Passanger, Staff, Driver, Steward  
- entity.train → Locomotief, Class373, Class374, Wagon, Train  
- entity.trip → Trip  
- entity.ticket → Ticket  
- ui → Main (startpunt van de applicatie)
